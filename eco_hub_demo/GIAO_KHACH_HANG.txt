EcoHub — hướng dẫn giao cho máy khách hàng
==========================================

1) Cài đặt
----------
- Giải nén (nếu nhận file .zip) hoặc copy cả thư mục chứa EcoHub.exe lên máy khách.
- Không cần cài Python.

2) Chạy lần đầu
---------------
- Double-click: EcoHub.exe
  (hoặc shortcut trỏ tới EcoHub.exe trong cùng thư mục.)
- Trình duyệt thường tự mở http://127.0.0.1:5000 (mặc định khi chạy bản .exe).
  Nếu không mở: mở trình duyệt và gõ địa chỉ hiện trên cửa sổ console.

3) Cấu hình (.env)
-------------------
- Đổi tên file .env.example thành .env (hoặc copy) và đặt CẠNH EcoHub.exe.
- Điền các biến ECOHUB_* / TikTok theo hướng dẫn trong .env.example.
- Không gửi file .env chứa mật khẩu thật qua kênh công cộng không mã hóa.

4) Dữ liệu trên máy khách (sau khi đã chạy)
-------------------------------------------
- Thư mục dữ liệu (bản .exe): cùng cấp với EcoHub.exe, thư mục con tên "data"
  (ví dụ: ...\EcoHub\data\). Trong đó có: cấu hình đã lưu, DB đăng nhập, TikTok token DB,
  thư mục "videos" (video local), "logs"…
- Muốn lưu sang ổ khác: trong file .env đặt biến ECOHUB_DATA_DIR=đường_dẫn_tuyệt_đối
  (thư mục đó phải có quyền ghi).
- Backup / chuyển máy: copy cả thư mục chứa EcoHub.exe (giữ nguyên "data" và "_internal").

5) Trong thư mục cùng EcoHub.exe có gì
---------------------------------------
- EcoHub.exe + thư mục _internal (PyInstaller 6): chứa thư viện, templates, static, config.json mặc định.
- Thư mục "data": dữ liệu runtime do app tạo (có thể đã có sẵn data\videos trong bản zip).
- Không được xóa hay tách _internal khỏi EcoHub.exe.
- File .env.example và bản hướng dẫn này (sau khi chạy build_exe.bat).

6) Chế độ đơn hàng (TikTok vs Shopee/khác chưa có API)
-------------------------------------------------------
- Mặc định: ECOHUB_ORDER_PLATFORM=tiktok — gọi API TikTok, đọc âm thanh chi tiết đơn.
- Đặt ECOHUB_ORDER_PLATFORM=generic (hoặc shopee, lazada, other, manual) — chỉ dùng mã quét,
  không gọi TikTok; video vẫn lưu theo mã đơn; âm báo ngắn (beep) thay cho TTS đơn hàng.
- Nhãn cột "Nền tảng": ECOHUB_GENERIC_ORDER_PLATFORM (vd SHOPEE).

7) Âm thanh đọc số lượng (TTS — chỉ khi TikTok)
-----------------------------------------------
- Cần đủ file .mp3 trong static/audio/tts/ (đã nằm trong bundle nếu có trong mã nguồn lúc build).
- Nếu không nghe: kiểm tra thư mục static/audio/tts trong thư mục cài đặt.

8) Hỗ trợ
---------
- Giữ lại cửa sổ console (màn hình đen) khi báo lỗi để chụp ảnh gửi bên triển khai.

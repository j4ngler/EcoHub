🔨 HƯỚNG DẪN BUILD ECOHUB QR SCANNER

📋 YÊU CẦU:

1. Python Dependencies

pip install -r requirements.txt
pip install pyinstaller>=6.0.0

2. Inno Setup (Tạo installer)

  • Download: https://jrsoftware.org/isdl.php
  • Cài đặt vào: C:\Program Files (x86)\Inno Setup 6\

🚀 CÁCH BUILD:

Bước 1: Build file .EXE

# Windows (Tự động):
build_exe.bat

# Hoặc manual:
pip install pyinstaller
pyinstaller build.spec --clean

Script sẽ: 1. Check PyInstaller 2. Install dependencies 3. Clean old build folders 4. Build EXE với PyInstaller

Kết quả: - File EXE: dist\EcoHub_QR_Scanner\EcoHub_QR_Scanner.exe (19 MB) - Internal: dist\EcoHub_QR_Scanner\_internal\ (200+ MB) - ✅ static/ folder - ✅ templates/ folder - ✅ config.json (default) - ✅ config.key (encryption key) - Python runtime + dependencies

Tổng kích thước: ~220-250 MB

Bước 2: Test EXE

cd dist\EcoHub_QR_Scanner
EcoHub_QR_Scanner.exe

Bước 3: Build Installer Setup

# Windows (Tự động):
build_installer.bat

# Hoặc manual:
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" setup.iss

Script sẽ: 1. Check Inno Setup installation 2. Check EXE exists 3. Compress tất cả files (LZMA) 4. Build installer

Kết quả: - File installer: installer_output\EcoHub_QR_Scanner_Setup_v1.0.0.exe - Kích thước: ~88 MB (compressed từ 220 MB) - Người dùng chỉ cần double-click để cài đặt!

⚙️ TÙY CHỈNH:

Đổi tên app:

Sửa trong setup.iss:

#define MyAppName "Tên App Của Bạn"
#define MyAppVersion "1.0.0"

Thêm icon:

  1. Tạo file icon.ico (256x256)
  2. Sửa build.spec: icon='icon.ico'
  3. Sửa setup.iss: SetupIconFile=icon.ico

Ẩn console:

Sửa build.spec: console=False (chỉ hiện GUI)
